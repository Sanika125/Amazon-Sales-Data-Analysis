# Amazon Sales Data Analysis 📊

## Project Overview

This project performs end-to-end sales data analysis using Python.

## Technologies Used

* Python
* Pandas
* Matplotlib

## Features

* Data Cleaning
* KPI Calculation
* State-wise Sales Analysis
* Category-wise Sales
* Monthly Revenue Trend
* Business Insights

## Key KPIs

* Total Revenue
* Total Orders
* Average Order Value

## Visualizations

* Top 10 States by Revenue (Bar Chart)
* Monthly Revenue Trend (Line Chart)

## How to Run

1. Install requirements
   pip install -r requirements.txt
2. Run main file
   python main.py



\## Dataset

The dataset is not uploaded due to size limitations.

You can download the dataset from Kaggle or use your own sales dataset.
